# Lua

## 構文

### 改行
文の区切りは改行でも `;` でも可。

### コメント
コメントは下記の2つの書き方がある。

```lua
-- 文字列出力
--[[
    コメント
]]
```

### 変数
変数は `名前 = 値` で宣言する。

```lua
message = "hello world"
```

多重代入も可能
```lua
hoge, fuga = hoge, fuga
```

#### 文字列
文字列はダブルクォーテーションで囲む。  
その中で特殊文字を使えば、改行やタブの挿入ができる。
```
s = "123\n456"
print(s)
```

下記で特殊記号がそのまま出力される。  
```
[[文字列]]
```
